Explain inline , internal and external stylesheet.

Inline:

In inline css we add a particular style to an element individually.We write inline css inside the tag using the attribute style.This CSS type is not really recommended, as each HTML tag needs to be styled individually.They’re useful for quick and permanent changes but are less flexible than external and internal stylesheets, as each inline style you create must be edited separately if you’re making a design change.

Internal:

In internal CSS we write the CSS properties in the head section of the html file.All the CSS styling properties are given under the style tag.All the changes done by the internal CSS can be applied only to a single web page. You can use internal CSS for one-page websites or projects where you do not want to use external CSS.

External:

In external CSS,we create a new file ending with .css extension.We link that file to the html file in which the we need to apply the styling properties.The linking is done in the head section of the html file using the link tag.We provide the styling properties by using the selectors.It has more advantageous to use this type of CSS.

What is css box model?

The CSS box model is a container that contains multiple properties including borders, margin, padding, and the content itself. It is used to create the design and layout of web pages.

There are several properties in the CSS box model. They are as mentioned below:

Content:

The content area consists of content like image, text, or other forms of media content. The height and width properties help to modify the box dimensions.

Padding:

The padding area is the space around the content area and within the border-box. It can be applied to all sides of the box or to the specific, selected side-top, right, bottom and or left.

Border:

The border area surrounds the padding and the content, and can be applied to all the sides of the box or to selected side- top, right, bottom or left.

Margin:

The margin area consists of space between the border and the margin. The margin does not possess its own background color and is completely transparent. It shows the background color of the element, like the body element.

What is the difference between padding and margin?

PADDING MARGIN
Padding controls the area immediately around your element. Margins control the spacing width between the edges of your web page and the element itself.
We can’t use negative and auto values for paddings. Margin supports negative as well as auto values.
It consists of padding-top:, padding-right:, padding-bottom:, and padding-left:. The property consists of margin-top:, margin-right:, margin-bottom:, and margin-left:.

What is the z-index in CSS?

z-index is a CSS property that defines the order of overlapping HTML elements. Elements with a higher index will be placed on top of elements with a lower index.

For example:-An element with z-index 2 will be placed on top of an element with z-index 1.
